# Memory
Création d'une suite de fichiers pour garder la mémoire des conversations , des sujets traités à chaque instance avec l'assistant copilot.
La limitation de caractères dans le champs de saisie, le nombre limité de réponses à 30 par instance, le fait que l'assistant ne peut pas ouvrir directement les fichiers pour lire le code ou pouvoir les modifier dans Github, rendent la tache compliquée car il faut copier-coller à chaque instance pour une continuité.
le copier-coller prend déjà 7 réponses sur 30. Nous travaillons avec l'assistant pour cela reste fluide, conçis, en tenant compte de toutes ses limitations.
je lui soumet des idées et on met à jour les fichiers.
Chaque instance , je lui rappelle les instances précédentes grace à la nomenclature qu'il m'a demandé de respecter pour que nos échanges restent logique.
L'objectif est que ,à terme, l'assistant est une sorte de mémoire continue.
